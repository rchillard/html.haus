API

Using the primary API, you can place a request as such: 

      
        curl https://html.haus/elements/abbr.json

You will receive a result that looks like this:

      
        {
          "name": "abbr",
          "description": "Abbreviation element represents an acronym",
          "code": "https://html.haus/elements/abbr.html",
          "url": "https://html.haus/reference.html#abbr"
      }

You can use references to code to also request the sample HTML:

      curl https://html.haus/elements/abbr.html

JavaScript

Ultimately, this means you can use JavaScript fetch functions to query the API using this formula

      
        https://html.haus/elements/${element-name}.json
        https://html.haus/elements/${element-name}.html

Here's two useful examples from Chris Ferdinandi:

JSON

      
        fetch('https://html.haus/elements/${element-name}.json').then(function (response) {
          // The API call was successful!
          return response.json();
        }).then(function (html) {
          // This is the JSON from our response
          console.log(html);
        }).catch(function (err) {
          // There was an error
          console.warn('Something went wrong.', err);
        });

HTML

      
        fetch('https://html.haus/elements/${element-name}.html').then(function (response) {
          // The API call was successful!
          return response.text();
        }).then(function (html) {
          // This is the HTML from our response as a text string
          console.log(html);
        }).catch(function (err) {
          // There was an error
          console.warn('Something went wrong.', err);
        });
