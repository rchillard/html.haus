Tools

Basic

These tools help you validate your HTML and write it better, faster, and stronger.

  • HTML Markup Validator (W3C)
  • Nu Html Checker (W3C)
  • Which Tag, When? Logic Tree
  • HTMLShell, online editor
  • Prepare Text for Pasting into HTML

Advanced

For building applications using an HTML first approach, these tools will simplify your life:

  • htmx, high power tools for HTML
  • hyperscript, language for interactive HTML
  • Hyperview, hypermedia format for server-driven mobile apps

Additional

For a background on HTML and Hypermedia, we highly recommend the Hypermedia Systems book by Carson Gross, Adam Stepinski, and Deniz Akşimşek.
