HTML Reference

Introduction

This is a catalog of HTML tags, which can be used for reference. Hover over HTML in the previous sentence to see an example. As part of this introduction, I just showed you how to use the abbr or abbreviation tag.

So, why is the above in a textarea tag? Well, textarea is a raw text element, meaning it displays whatever appears inside of it as raw or plain text, even including HTML tags. The nice part is that you can display HTML tags inside of a textarea with very little extra work. This is important for demonstration and teaching persons. It's also easy to copy and paste. I invite you to do both!

This reference makes heavy use of the <textarea> tag in order to show raw HTML.

Types of Tags

Other HTML references focus on categorizing tags as normal or self closing. This one does not. Since example code is provided with each element/tag, I think it's self evident which tags are normal (with opening and closing tags) and which are self closing tags. In cases where this really matters, the description will specify which type of tag the element uses.

Global Attributes

There are common HTML attributes that apply to every element. This list is omitted from the reference for individual elements, because it would be incredibly duplicative and you should know them anyway. Common or global HTML attributes:

  • class, used to group elements together, as well as attach CSS styles to them collectively
  • data-*, allows you to attach custom data to an HTML element, opening a world of possibility
  • draggable, defines whether an element can be dragged by the cursor or mouse
  • hidden, specifies whether the element should be visible or not
  • id, used to identify individual elements, mostly for the purpose of attaching CSS styles
  • inputmode, allows you to specify a hint for the end user editing this element
  • style, used to apply in line CSS styles, as the example above for style="color: red"
  • tabindex, allows you to specify whether an element should be focusable and when
  • title, allows you to specify extra information about an element
There are more global attributes, but these are the most common that you'll use widely and often.

Types of Elements

HTML is divided into different types of elements, each family of which serves a purpose in structuring content for the user. Some elements organize, Some elements denote, some add tabular structures, while others allow you to submit data or interact with the page. Here are the different kinds of elements:

  • Sectioning
  • Text content

Attributes

  • href, hypertext reference, or address
  • target, keyword or defined name of the default browsing context within which to display the result of links or forms
    • _self: Load the result into the same browsing context as the current one. (This is the default.)
    • _blank: Load the result into a new, unnamed browsing context.
    • _parent: Load the result into the parent browsing context of the current one. (If the current page is inside a frame.) If there is no parent, behaves the same way as _self.
    • _top: Load the result into the topmost browsing context (that is, the browsing context that is an ancestor of the current one, and has no parent). If there is no parent, behaves the same way as _self.
